# Titanic - Machine Learning from Disaster
## A Kaggle challenge to figure out "what sorts of people were more likely to survive?" using passenger data (name, age, gender, ...)

### Download dataset
```bash
kaggle competitions download -c titanic
```

### Overview
There are two sets of data:
* training set (train.csv)
* testing set (test.csv)

[Learn more](https://www.kaggle.com/competitions/titanic/data).

---
We are supposed to make predictions for testing set and submit the csv file.
With no prior ML knowledge I referred to kaggle's recommended tutorial: [Alexis Cook's Titanic Tutorial](https://www.kaggle.com/code/alexisbcook/titanic-tutorial). And the code is identical to one provided in tutorial.
